# Texas Hold'em Multiplayer Poker

Private room Texas Hold'em — no login required.

## Quick Start

```bash
# Terminal 1 — backend
cd server && npm install && npm run dev

# Terminal 2 — frontend
cd client && npm install && npm run dev
```

Open http://localhost:5173 in two browser tabs to test multiplayer.

## Rules Supported
- No-Limit Texas Hold'em
- Custom starting chips, blinds, max players
- Automatic hand evaluation + winner detection
- Reconnection support

## Not in v1 (planned for later)
- User accounts / login
- Database persistence
- Virtual currency / leaderboards
- Observer mode
